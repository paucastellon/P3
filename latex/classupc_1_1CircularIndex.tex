\doxysubsection{Detailed Description}
Circular Index used to go through state circular buffer (u) 

The circular index is a simple class to address a circular buffer. Once the buffer size is defined, you can add or substract and the index remains inside the buffer. A casting operator to int is provided.

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{float} x[8]=\{0,1,2,4,5,6,7\};}
\DoxyCodeLine{\mbox{\hyperlink{classupc_1_1CircularIndex_a80c03ec94380d80132ca01b39ef9b0e7}{CircularIndex}} ci(8);}
\DoxyCodeLine{\textcolor{keywordtype}{int} n=10; \textcolor{keywordtype}{float} z=0.0F;}
\DoxyCodeLine{\textcolor{keywordflow}{for} (n=0; n< 10; ++n, ++ci)}
\DoxyCodeLine{    z += x[ci+1] -\/ x[ci-\/1];}

\end{DoxyCode}
 Operations +, +=, ++, -\/, -\/=, -- are defined
